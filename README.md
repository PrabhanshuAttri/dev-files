# vim
My vim settings, terminal settings and installer

## Usage

- vimrc

  > cp vimrc ~/.vimrc
  
  > git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
  
  > vim +PluginInstall

- bash_profile

- terminal_settings.terminal

- sweeper

  > chmod +x sweeper
  
  > ./sweeper
